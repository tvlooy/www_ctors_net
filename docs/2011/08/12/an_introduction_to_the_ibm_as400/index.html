An introduction to the IBM AS/400

IBM i | August 12, 2011

I spoke at HAR2009.

At the Hacking At Random (HAR) conference, I did a presentation about IBM's AS/400 system. The talk was intended for people that don't know AS/400. Or, just want to know what's so special about it and want to know how it works.

The talk gave a quick historical overview of this extraordinary system, followed by a technical introduction of what defines the IBM i:

  • technology independent machine interface
  • integrated database
  • object based design
  • single level store

Next, the basics of working with the system were covered.
